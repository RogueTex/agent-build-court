# Run 07: Practical Pivot

## User Correction

The hackathon rules were constraints, not the product. The initial submission-readiness build was too meta.

## Grill-Me Reframe

Question: what practical app can be built in 15-20 minutes, use spec-first docs, make agent committee structure visible, and still satisfy public no-login submission requirements?

## Candidate Themes

- Interview Prep Sprint Coach
- Agent committee interface
- Resume bullet rewriter
- Offer decision matrix

## Vote

Final direction: Agent Huddle.

Reason: it makes the "agents talking to agents" idea concrete. It is useful for Codex users, requires no login or backend, and directly demonstrates an agent committee prompt structure without pretending to run hidden AI in the browser.

## MVP Scope

- Brief input: goal, timebox, output format, constraints, assets.
- Four visible agents: Product, Builder, Skeptic, Judge.
- Deterministic transcript from the shared brief.
- Ship/revise vote and risk count.
- Copyable prompts for running the same committee as Codex subagents.

## Non-Goals

- No live AI API.
- No auth.
- No database.
- No private integrations.
- No fake claims that browser agents are autonomously reasoning.
