# Run 08: Winning Debate

## Goal

Strengthen Agent Huddle into something that can win a hackathon.

## Agent Debate

Three agents reviewed the product:

- Hackathon Judge: rejected the basic huddle as too much like a prompt formatter.
- Product/UX Agent: recommended "prompt war room" energy with visible disagreement and vote.
- Build Lead: recommended rounds, conflict map, handoff queue, and Markdown/JSON export.

## Decision

Ship Agent Build Court.

The winning version puts a messy idea on trial and produces a real artifact, not just advice.

## Added To MVP

- Three debate rounds: opening arguments, cross-examination, verdict.
- Judge scorecard.
- Conflict map.
- Handoff queue.
- Copy/download-ready `BUILD_PACKET.md`.
- JSON export.
- Updated spec, judging notes, and submission draft.

## Non-Goals

- No live AI API.
- No auth.
- No database.
- No drag-and-drop.
- No persistence.
- No claim that browser agents are autonomously reasoning.
