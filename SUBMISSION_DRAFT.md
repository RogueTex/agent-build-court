# Codex Hackathon Submission Draft

## Project Title

Agent Huddle

## What Did You Build?

Agent Huddle is a no-login static app that turns a messy goal into a structured four-agent committee transcript. It shows product, builder, skeptic, and judge perspectives, gives a ship/revise vote, flags scope risks, and exports copyable Codex subagent prompts. The public demo works as plain HTML/CSS/JS with no backend or account setup.

## How Did You Use Codex?

Codex served as the build lead. It rejected the earlier meta submission-checker, grilled candidate MVPs, voted for Agent Huddle, wrote the spec-first plan, generated the agent committee prompt structure, implemented the HTML/CSS/JS app, updated the docs, and verified the repository smoke test.

## Current Status

Working static MVP. The app has no backend, login, database, dependency install, or live AI call. Committee output is deterministic and honest; the exported prompts can be pasted into Codex to run the real agent committee.
